# PlotStudio AI Demos

AI-generated analyses by [PlotStudio AI](https://www.plotstudio.ai/?utm_source=github&utm_medium=repo&utm_campaign=demos). Each analysis was produced from a single question — the AI planned the approach, wrote the code, executed it, and interpreted the results.

PlotStudio is an agentic analytics tool where AI agents investigate your data. Upload a dataset, ask a question, and the agents plan the analysis, write and execute Python code, generate visualizations, and deliver a complete report with interpretation.
