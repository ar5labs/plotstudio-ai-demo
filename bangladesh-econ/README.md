## Key Findings

- No hard over-investment cutoff exists within Bangladesh's observed spending range (0.9%-2.2% of GDP)
- Diminishing returns begin around 1.1% of GDP: below that, education spending has a steep positive association with GDP per capita growth; above it, the marginal payoff flattens sharply
- The relationship weakened significantly after 2003 — the apparent threshold is partly a development-stage effect, not a fixed spending ceiling
- Granger tests found no predictive precedence in either direction; the link is contemporaneous association, not a lagged causal channel
- Cointegration was not supported (p = 0.78), so the analysis correctly used differenced series throughout
